# Hackathon Todo CLI

**Version:** 1.0.0  
**Author:** Eman Kk  

## Overview
Hackathon Todo CLI ek simple command-line tool hai jo tasks ko manage karne ke liye banaya gaya hai.  
Ye tool developers, students, aur hackathon participants ke liye perfect hai jo quickly tasks add, update, delete aur mark complete/incomplete karna chahte hain.

## Features
- Add new tasks  
- List all tasks  
- Update existing tasks  
- Delete tasks  
- Mark tasks as complete or incomplete
